# 🎭 Dramatic Advice Generator (AI-Powered Web Application)

🔗 **Live Application:**  
👉 https://dramatic-advice.vercel.app/

---

## 🚀 Overview

Dramatic Advice Generator is a full-stack AI-powered web application that generates mood-based dramatic advice using a generative AI API.

This project demonstrates production-level backend development using Spring Boot, secure AI integration, Docker containerization, and cloud deployment with proper frontend-backend communication.

---

## 🧠 Features

- ✨ Generate AI-powered advice based on mood and user input  
- 🔥 RESTful APIs built using Spring Boot  
- 🏗 Clean layered architecture (Controller → Service → DTO)  
- 🔐 Secure AI API integration using environment variables  
- 🌍 Production-ready CORS configuration  
- 🐳 Dockerized backend deployment  
- ☁️ Cloud-hosted backend (Render) and frontend (Vercel)  
- ⚙️ Environment-based configuration (Local vs Production)  
- 🔑 Secure API key management  

---

## 🏗️ Backend Architecture

The backend follows a clean layered architecture:

- **Controller Layer** – Handles HTTP requests and responses  
- **Service Layer** – Contains business logic and AI integration  
- **DTO Layer** – Manages request/response payload mapping  
- **Configuration Layer** – Handles CORS and environment setup  

### Core Backend Capabilities

- REST API design using Spring Boot  
- JSON request/response mapping  
- Exception handling and error management  
- Cross-origin configuration for cloud deployment  
- Docker-based containerization  
- CI/CD integration via GitHub
